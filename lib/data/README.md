# 数据

信息 → 编码 → 数据 → 计算机 → 数据 → 解码 → 信息

形式 := 信息 | 数据 | 符号
形式转换 := 编码 | 解码 | 转换

## 数据结构

### 逻辑结构

D = (E, R).

E 是数据结构 D 的元素集合(有穷集), R ∈ E x E 是元素之间的关系.

由 R 导出,
D 的结构 := 集合结构(空) | 线性结构(一对一) | 树形结构(一对多) | 图形结构(多对多).

### 存储结构

顺序存储 := 数组

链式存储 := 链表

散列存储 := 散列

索引存储 := 索引表 + 数组/链表

### 数据运算

数据运算的定义(逻辑结构) := 算法设计

数据运算的实现(存储结构) := 算法实现

## 数据类型

数据类型 := 数据集合 + 操作集合

### Ruby 的类

数组 → 线性表/栈/队列

散列 → 哈希表

结构体 → 链表/树

二维数组/散列 → 图
